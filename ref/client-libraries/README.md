# Reference - Client Libraries

## Web API

The following client libraries are available for the Web API.

| Language | Library | Documentation |
| - | - | - |
| C# | [`Primavera.IdentityServer.dll`](http://nuget.primaverabss.com:82/feeds/public-lithium-general/Primavera.IdentityServer) | [5.0](https://dv-ids.lithium.primaverabss.com/.doc/clientlib) |

## Core Endpoints

The core endpoints can be consumed directly and/or using any client library compatible with OAuth 2.0 and OIDC.

PRIMAVERA Hydrogen includes two types designed to consume these endpoints:

- `Primavera.Hydrogen.IdentityModel.Client.DiscoveryClient`
- `Primavera.Hydrogen.IdentityModel.Client.TokenClient`
